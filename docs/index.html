Making Firefox session files legible and searchable

Firefox stores session information in .jsonlz4 files which are 100% not human-readable. On a Mac, these files can be found in the Firefox profile directory: 

~/Library/Application Support/Firefox/Profiles/*profile name*/sessionstore-backups/recovery.jsonlz4
~/Library/Application Support/Firefox/Profiles/*profile name*/sessionstore.jsonlz4

Method 1 (two-step decompression and html conversion)

Python code (not my code) to convert Firefox saved session information (now in recovery.jsonlz4) to html.
  • Step 1 is now to uncompress the Firefox .jsonlz4 file (compressed in a weird LZ4 format, for reasons known only to Mozilla) to a normal json file. One way to do this is with the firefox extension "mozlz4-edit" which will export to a suitable text format via "Save as JSON". Another way is to use mozlz4a.py: 
     $ python3 mozlz4a.py -d recovery.jsonlz4 recovery_json.txt

  • Step 2 is to convert the resulting text (JSON) file to human-readable HTML: 
    $ python session2html.py -s recovery_json.txt -o recovery-pagelist.html

@mirogula's session2html was based in turn on @warner's make-html.py. These earlier codes worked on the uncompressed JSON files as produced by earlier versions of Firefox.

Method 2

I wrote a revised Python script to do the .jsonlz4 decompression and HTML output in one step (adapting code from session2html and following the example in @tblue's and @kaefer3000's mozlz4a.py for the decompression step). Use python3 for this version: 

$ python3 session2htmlp3.py -s recovery.jsonlz4 -o recovery-pagelist.html
All this assumes you've gotten a copy of your python code and a copy of the .jsonlz4 file in the same directory - otherwise you'll have to include the file paths.

Music

Here is the viola part to Mozart's "Kegelstatt" trio (K. 498) transcribed for B-flat clarinet as well as treble clef. Basically, some friends and I wanted to play this piece, which was written for violin or clarinet, viola, and piano, but what we actually had was flutes, a clarinet, and a piano, hence the transcription.

Viola part, movement 1, for B-flat clarinet (Andante): .pdf format

Viola part, movement 2, for B-flat clarinet (Menuetto/Trio): .pdf format and .pdf including the rehearsal letters from the Schirmer edition.

Viola part, movement 3, for B-flat clarinet (Allegretto): .pdf format

Viola part, movement 1, in treble clef (Andante): .pdf format

Viola part, movement 3, in treble clef (Allegretto): .pdf format

Note that the original public domain parts are available on Mutopia as are Lilypond source files of original parts. (Lilypond is kind of like LaTeX for music typesetting.)
